# Finance-AI-Automation-Comparative-Intelligence-Platform

This project automates financial and document analysis by standardizing and comparing multiple financial statements and PDFs.
It identifies low-rated or inconsistent data, suggests improved structures, extracts relationships and sequences using embeddings, and enables question answering across multiple uploaded documents using a RAG-based architecture.
The system supports comparative analysis, validation, and insight generation, and is designed to handle messy real-world data rather than clean academic datasets.
Vectara Financial Analysis Dashboard is a Streamlit app that enables intelligent search and comparison over financial PDF documents using Vectaraâ€™s RAG pipeline. Users can connect via API key, upload one or more PDFs to a Vectara corpus, ask natural-language questions, view source snippets with relevance scores, and generate a metrics comparison table (Revenue, Net Profit, Assets, etc.) with export to CSV and basic visualization.
